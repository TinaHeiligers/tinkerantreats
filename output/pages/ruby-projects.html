Tutorials with Code

6 Ruby Best Practices Beginners Should Know is a fantastic read, concise and gives code to demonstrate the ideas.

For example, avoiding loops in Ruby: Don't do ``` for elem in [1, 2, 3, 4, 5] puts elem end

Do do:

[1, 2, 3, 4, 5].each do |elem| puts elem end

Or even better:

[1, 2, 3, 4, 5].each do { |elem| puts elem } ```
